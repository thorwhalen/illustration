illustration.providers

Built-in provider sources, registered on import.

Importing illustration imports this package, which instantiates and registers each built-in RetrievalSource. A third-party provider registers itself the same way: illustration.register_source(MySource()).

(This subpackage is named providers rather than sources so it does not shadow the public illustration.sources registry view.)
