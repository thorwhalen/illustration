illustration.providers.openverse

Openverse source — 800M+ openly-licensed / public-domain images, no key.

Openverse (https://api.openverse.org/v1/images/) aggregates Creative-Commons and public-domain media from Flickr, Wikimedia, museums, and more. The anonymous tier needs no credentials, which makes it the package’s out-of-the-box default (illustration.search("...") works with zero setup). Every result carries a license code, a canonical license URL (except CC0/PDM, where it is None), and a ready-to-render attribution sentence — so licensing is populated from day one.

Quirks handled here (from the live-API survey): tags are objects ({"name": ...}), url is the real image while thumbnail is an Openverse proxy, and the anonymous tier caps page_size at 20.
