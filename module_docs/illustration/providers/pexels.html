illustration.providers.pexels

Pexels source — high-quality stock photos, API key required.

Pexels (https://api.pexels.com/v1/search) is the package’s reference keyed provider: it proves the registry is a true drop-in and exercises the credentials / check_requirements path. The key is sent as the raw value of the Authorization header (no Bearer prefix — a prefix yields 401).

Quirks handled here (from the live-API survey): photos[].url is the Pexels web page, not the image (the image is under photos[].src.*); there are no tags and no per-result license (all content is the single Pexels License).
